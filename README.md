# 📚 BookNest

BookNest is a modern full-stack bookstore web application built using the MERN stack.

It provides a complete online bookstore experience with authentication, cart management, wallet system, order tracking, reviews, recommendations, admin dashboard, and responsive UI.

---

# 🚀 Features

## 👤 User Features

- User Authentication (JWT)
- Add to Cart
- Wishlist
- Buy Books
- Wallet System
- Order Tracking
- Recently Viewed Books
- Reviews & Ratings
- Recommended Books
- Search Books
- Bestseller Books
- Responsive UI

---

## 🛠️ Admin Features

- Add Books
- Update Books
- Delete Books
- Manage Orders
- Update Order Status
- Bestseller Management
- Category Management
- Stock Management

---

# 🧑‍💻 Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
